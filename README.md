## 1. Деплой на хостинг (Netlify)

Netlify — бесплатный хостинг для статических сайтов, идеально подходит для этого проекта.

### Шаг 1: Создай аккаунт
Перейди на [netlify.com](https://netlify.com) и зарегистрируйся (можно через GitHub или Google).

### Шаг 2: Загрузи сайт
1. На главной панели найди раздел **"Sites"**
2. Прокрути вниз до блока **"Deploy manually"**
3. Перетащи папку `wedding-invitation-main` прямо в выделенную зону
4. Подожди несколько секунд — сайт развернётся автоматически
5. Netlify выдаст временный адрес вида `random-name-123.netlify.app`

> Проверь, что сайт открывается по этому адресу перед продолжением.

## 2. Покупка домена

### Где купить
Рекомендую один из этих регистраторов:

| Регистратор | Сайт | Примечание |
|---|---|---|
| **Namecheap** | [namecheap.com](https://namecheap.com) | Удобный интерфейс, есть бесплатный WhoisGuard |
| **REG.RU** | [reg.ru](https://reg.ru) | Популярен в России, поддержка на русском |
| **Cloudflare** | [cloudflare.com/products/registrar](https://www.cloudflare.com/products/registrar/) | Продаёт домены по себестоимости, встроенный DNS |

### Как купить (на примере Namecheap)
1. Перейди на [namecheap.com](https://namecheap.com)
2. В строке поиска введи желаемое имя домена, например `vasya-i-masha.com`
3. Выбери подходящий домен и нажми **Add to cart**
4. Оформи заказ, укажи данные и оплати
5. Домен появится в разделе **Domain List** в личном кабинете

> Домены `.com` стоят ~$10–15/год, `.ru` — ~300–500 ₽/год.

## 3. Добавление DNS-записи

Нужно привязать домен к Netlify. Netlify выдаст тебе IP-адрес или CNAME, которые нужно добавить у регистратора.

### Шаг 1: Добавь домен в Netlify
1. В Netlify перейди на страницу своего сайта
2. Нажми **Domain settings → Add a domain**
3. Введи свой домен, например `vasya-i-masha.com`, и нажми **Verify**
4. Netlify покажет нужные DNS-записи — **скопируй их**

Обычно Netlify попросит добавить два типа записей:
- **A-запись** для корневого домена (`@`) → IP-адрес Netlify (например `75.2.60.5`)
- **CNAME-запись** для `www` → `[имя-сайта].netlify.app`

### Шаг 2: Добавь записи у регистратора (на примере Namecheap)
1. Войди в [namecheap.com](https://namecheap.com) → **Domain List**
2. Нажми **Manage** рядом со своим доменом
3. Перейди на вкладку **Advanced DNS**
4. Нажми **Add New Record** и добавь:

**A-запись:**
```
Type:  A Record
Host:  @
Value: 75.2.60.5
TTL:   Automatic
```

**CNAME-запись:**
```
Type:  CNAME Record
Host:  www
Value: [имя-сайта].netlify.app
TTL:   Automatic
```

5. Нажми **Save All Changes**

> DNS обновляется от 5 минут до 48 часов. Обычно — в течение 15–30 минут.

### Проверка
Когда DNS распространится, открой свой домен в браузере — должен открыться сайт.

## 4. Подключение HTTPS

Netlify подключает HTTPS **автоматически и бесплатно** через Let's Encrypt — никаких дополнительных действий не нужно.

### Что делает Netlify автоматически:
- Выпускает SSL-сертификат через Let's Encrypt
- Автоматически обновляет сертификат каждые 90 дней
- Перенаправляет HTTP → HTTPS

### Как проверить статус:
1. Перейди в Netlify → **Domain settings → HTTPS**
2. Дождись статуса **"Your site has HTTPS enabled"** (зелёная галочка)
3. Это может занять 5–30 минут после настройки DNS

### Если HTTPS не появился:
- Убедись, что DNS-записи настроены правильно
- Нажми **Verify DNS configuration** в Netlify
- Нажми **Renew certificate** если прошло больше часа
